# Restaurant Website

## Live Demo

[Open Website](https://glittering-griffin-96bfd7.netlify.app/)

## Features

* Responsive modern design

* Contact page
* Blog section
* Customer reviews section

## Technologies Used

* HTML5
* CSS3
* JavaScript

This project was created to practice front-end web development skills and website structure design.
